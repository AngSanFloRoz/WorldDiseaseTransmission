# WorldDiseaseTransmission 
Simulación visual de propagación viral sobre un grafo — HTML + JavaScript

Virus Spread Simulator es una herramienta interactiva que muestra cómo un virus se expande dentro de una red (grafo).
El proyecto funciona completamente en el navegador, sin dependencias externas, utilizando únicamente:

HTML para la estructura

CSS para el diseño

JavaScript para la lógica de simulación y renderización

Este proyecto es ideal para demostraciones, clases de estructuras de datos, teoría de grafos, epidemiología básica o visualizaciones interactivas.

# Características
Simulación dinámica

Infecta un nodo inicial y observa cómo evoluciona el contagio paso a paso.

La interfaz visual actualiza los estados de los nodos en tiempo real.

# Visualización clara y directa

Cada nodo cambia de color según su estado (por ejemplo, saludable, infectado).

Las conexiones del grafo se dibujan directamente en el canvas con JavaScript.

# Proyecto 100% web

No requiere instalación.

Funciona directamente al abrir index.html en cualquier navegador moderno.

Ideal para alojar en GitHub Pages.
