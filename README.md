# WorldDiseaseTransmission 
Es una herramienta desarrollada para modelar, visualizar y analizar la expansión de un agente infeccioso dentro de una red (grafo).
El proyecto combina conceptos de teoría de grafos, simulación estocástica, dinámica de propagación y estructuras de datos para reproducir escenarios realistas y personalizables.

El objetivo principal es ofrecer una plataforma educativa y experimental, ideal para:

Estudiantes y docentes de computación, matemáticas y ciencias de datos.

Investigadores que necesiten prototipar comportamientos de propagación.

Desarrolladores interesados en simulaciones basadas en grafos.

Cualquier persona que quiera visualizar de manera sencilla cómo se expande un virus dentro de una red de conexiones.
